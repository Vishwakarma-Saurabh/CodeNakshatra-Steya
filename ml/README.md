# ML Scripts

This folder contains standalone scripts for testing and developing ML algorithms.

## Structure

- `scripts/` - Python scripts for testing algorithms
- `data/` - Sample data for testing
- `notebooks/` - Jupyter notebooks for experimentation

## Usage

### Test Distance Calculation
Run the distance test script to verify Haversine formula accuracy.

### Test Route Matching
Run the matching test script with sample route data.

### Generate Test Data
Create sample routes and rooms for testing the recommendation system.

## Note
The actual ML algorithms used by the application are located in:
`backend/app/services/ml_service.py`

This folder is for development and testing purposes only.
