# 🏠 Steya

Steya is a smart platform that helps people find flexible stays and room-sharing options near their destination.  
It also enables users to list unused rooms and optionally match with others for mutual stay benefits.

---

## 🚀 Problem Statement

People travel long distances daily for work, study, or other reasons.  
There is no efficient platform for:
- Short-duration stays (few hours / daily use)
- Affordable nearby rooms
- Matching users with opposite travel routes
- Utilizing unused rooms efficiently

---

## 💡 Solution

Steya solves this by providing:
- 📍 Nearby room discovery
- 🏠 Short-term stay booking (4–8 hours / daily)
- 🔄 Smart matching for possible room swaps
- 🛡️ Trust-based user system (ratings + verification)

---

## 🧠 Key Features

- Search rooms near destination
- List personal rooms for rent
- Short stay booking system
- Optional smart match system
- User profile & trust system

---

## 🏗️ Tech Stack

- Frontend: React / HTML / CSS / JavaScript
- Backend: Node.js / FastAPI
- Database: MongoDB / PostgreSQL (primary choice)
- ML : Python (recommendation system)
